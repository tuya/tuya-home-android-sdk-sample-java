Detailed Description

Mqtt send and receive channels.

Send messages to or receive messages from the device via MQTT.

This feature is only needed if you need to customize the embedded firmware, which will handle the corresponding mqtt messages.

ITuyaMqttChannel mqttChannelInstance = TuyaHomeSdk.getMqttChannelInstance();
mqttChannelInstance.registerMqttRetainChannelListener(new ITuyaMqttRetainChannelListener() {
    @Override
    public void onMessageReceived(MqttMessageBean messageBean) {
        int protocol = messageBean.getProtocol();
        JSONObject responseJsonObj = messageBean.getData();
    }
});
mqttChannelInstance.sendDeviceMqttMessage(devId, hashMap, 64, callback);

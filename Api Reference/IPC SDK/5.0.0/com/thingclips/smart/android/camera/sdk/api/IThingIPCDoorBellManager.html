         abstract void setIgnoreWhenCalling(boolean ignoreWhenCalling)

        When receiving a doorbell call from a device, set whether to automatically ignore subsequent calls from this device. Take effect for all equipment.

        Parameters:
        ignoreWhenCalling - True: If a device is receiving a doorbell call, it will automatically ignore subsequent callsfrom this device; false: Do not ignore subsequent calls from the same device.

      • setDoorbellRingTimeOut

         abstract void setDoorbellRingTimeOut(int doorbellRingTimeOut)

        Set the doorbell call timeout time. If the doorbell call is not processed within the specified time, it will call back doorBellCallDidCanceled to notify and end the doorbell call.

        Parameters:
        doorbellRingTimeOut - Doorbell call timeout time, in seconds, the default value is 25s.

      • generateCallModel

         abstract void generateCallModel(String deviceId, String messageType, String msgId, long startTime, String cid)

        In addition to notifying the client through MQTT protocol messages, doorbell calls may also be pushed,so this interface is provided. The interface can generate a doorbell call model based on external inputparameters to manage subsequent doorbell answering, hanging up, and rejection operations.

        Parameters:
        deviceId - Doorbell device id
        messageType - Doorbell call Message type
        msgId - The unique identifier of the doorbell call message
        startTime - Doorbell call start time
        cid - NodeId of the sub device (for example: gateway sub device), nullable
